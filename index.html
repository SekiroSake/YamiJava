Yami is an Android app that aims to use personalization to improve restaurant recommendation based on their visit history and preference

  • This is a runnable java version
  • Imported and parsed restaurant and user information from Yelp open dataset and Yelp API (JSON)
  • Built a personalized web service to store restaurant information and user visit history (MySQL, Java)
  • Deployed production service to Amazon EC2 cloud and tested the peak performance
  • Designed and implemented a filtering and sorting algorithm to recommend similar restaurants based on user's history and preference

how to run

  • Download and unzip the file
  • Import it to your eclipse or any SDK you prefer
  • Initiate the server and add it to server list
  • You may try to query and get result on https://chrome.google.com/webstore/detail/advanced-rest-client/hgmloofddffdnphfgcellkdfbfbjeloo?hl=en-US
